MCP Bridge 配置指南

📅 2026-05-26

📚 不了解 MCP？先看 MCP 协议基础

MCPBridge配置

MCP Bridge v2 是 Agent Mesh 网络的核心组件，提供跨 Agent 的 MCP 工具调用和上下文共享。

下载

从 GitHub 获取最新版本：

mkdir -p ~/.openclaw/mcp-bridge && cd ~/.openclaw/mcp-bridge
curl -sL https://raw.githubusercontent.com/frisky1985/yuleAI-Hub/main/tools/mcp-bridge/mcp_bridge_server.py -o mcp_bridge_server.py

安装

# 注册到 OpenClaw
openclaw mcp set mcp-bridge '{
  "command": "python3",
  "args": ["~/.openclaw/mcp-bridge/mcp_bridge_server.py"]
}'

# 验证
openclaw mcp list

MCP 工具列表

工具说明响应时间
agent-status检查所有 agent 是否在线<1s

task-result查询任务结果即时
task-list查看最近任务即时
context-read读取共享上下文即时
context-write写入共享上下文即时

故障排查

连接失败

检查 MCP 配置是否正确：

openclaw mcp show mcp-bridge

任务超时

增大 Bridge 的超时设置（编辑 mcp_bridge_server.py 中的 MAX_ASYNC_SEC）：

MAX_ASYNC_SEC = 600  # 10 分钟

Agent 找不到

确保 agent 在 PATH 中：

which hermes qodercli openclaw
