🔄 Agent Ping

Agent Mesh 保活脚本 — 定时检测 OpenClaw/Hermes/QoderCLI 在线状态

说明

30 分钟自动检测所有 Agent 在线状态，更新保活时间戳。配合 crontab 使用。

下载

curl -sL https://raw.githubusercontent.com/frisky1985/yuleAI-Hub/main/tools/agent-ping/agent-ping -o agent-ping
chmod +x agent-ping
# 放入 PATH
mv agent-ping ~/.local/bin/

安装 crontab

(crontab -l 2>/dev/null; echo "*/30 * * * * $HOME/.local/bin/agent-ping") | crontab -
# 验证
crontab -l | grep agent-ping

手动运行

agent-ping
