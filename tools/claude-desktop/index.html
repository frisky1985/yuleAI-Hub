💻 Claude Desktop

AI 编程助手 · macOS ARM64 · Apple Silicon

Claude Desktop 是 Anthropic 官方推出的桌面端 AI 编程助手 IDE。通过配置 DeepSeek 原生 Anthropic 端点，可在中国大陆正常使用。

⌘

Claude Desktop for macOS

ARM64 (Apple Silicon) · ~281 MB (zip)
⬇ 下载 Claude.dmg.zip

安装步骤

  1. 下载 Claude.dmg.zip 并解压得到 Claude.dmg
  2. 打开 Claude.dmg，将 Claude 拖入 Applications 文件夹
  3. 首次打开若提示"无法验证开发者"，右键 → 打开 即可

在中国使用

Claude Desktop 默认连接 Anthropic API，从中国大陆 IP 访问会被封锁。需通过 DeepSeek 原生 Anthropic 端点绕过：

详见 Claude Code + DeepSeek V4 配置指南。
