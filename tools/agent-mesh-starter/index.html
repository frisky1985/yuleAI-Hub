🚀 Agent Mesh 一键启动器

工具Agent MeshmacOS · 2026-05-27

双击即用的一键启动脚本。自动在 Warp 终端中拉起 MCP Bridge、Gateway 和所有 Agent，并进入 OpenClaw 对话界面。

📥 下载脚本

⬇ 下载 Start-Agent-Mesh.sh

macOS · 需安装 Warp 终端、OpenClaw、Hermes、QoderCLI

安装方式

1
下载后打开「终端」运行：
chmod +x ~/Downloads/start-agent-mesh.sh
2
移动到常用位置：
mv ~/Downloads/start-agent-mesh.sh ~/Desktop/
3
双击 start-agent-mesh.sh → 自动在 Warp 中打开并执行

或直接通过命令行运行：

bash ~/Desktop/start-agent-mesh.sh

启动流程

  [q] 退出

交互菜单

依赖

  • Warp 终端 — 自动唤起，提供更好的 TUI 体验
  • OpenClaw — 主 Agent + Gateway
  • Hermes — 子 Agent（需求分析、排期、测试）
  • QoderCLI — 子 Agent（架构设计、编码）
  • MCP Bridge — Agent 间通信桥梁
