Warp Agent 集成指南

2026-05-26

Warp终端配置

在 Warp 终端中配置和使用 AI Agent，以及解决常见问题。

Warp Agent 配置

Warp 的内置 Agent 通过 ~/.warp/settings.toml 配置：

[agents.warp_agent]
is_any_ai_enabled = true

[agents.warp_agent.input]
nld_in_terminal_enabled = true

[agents.third_party]
should_render_cli_agent_toolbar = true

在 Warp 中运行 Agent CLI

Warp 支持多种 CLI Agent 的增强工具栏（输入编辑、代码审查等）：

# 直接运行 hermes
hermes chat

# 或使用快捷方式
hermes-warp

解决 AI 功能被限制

如果看到 "Your account has been blocked from using AI features" 错误：

  • 使用 BYOK — Settings → Cloud Platform → Third-Party API Keys，添加你自己的 API Key
  • 升级付费 — 或联系 appeals@warp.dev 申诉
  • 忽略 — 直接用 hermes 或其他 CLI Agent，不影响 Warp 的终端功能

Agent Mesh 集成

在 Warp 终端中运行 OpenClaw，即可通过 MCP Bridge 调用其他 Agent：

# 在 Warp 中启动 OpenClaw
openclaw

# 然后在 OpenClaw 对话中
"用 agent-status 检查所有 agent 状态"
