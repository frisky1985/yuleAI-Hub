QoderCLI 配置与使用

2026-05-25

QoderCLICLI配置

QoderCLI 是字节跳动出品的 AI 编程助手，支持 MCP 协议集成。

基本使用

# 交互模式
qodercli

# 单次查询
qodercli -p "列出当前目录的文件"

# 跳过权限确认
qodercli -p "命令" --dangerously-skip-permissions

MCP 集成

QoderCLI 原生支持 MCP 服务器配置：

# 添加 MCP 服务器
qodercli mcp add-json mcp-bridge '{
  "command": "python3",
  "args": ["/path/to/mcp_bridge_server.py"]
}'

# 查看已配置的 MCP 服务器
qodercli mcp list

接入 Agent Mesh

在 QoderCLI 中可以直接调用 OpenClaw 和 Hermes：

# 在 QoderCLI 对话中
"帮我用 agent-chat 问一下 hermes 当前系统状态"

注意事项

  • QoderCLI 交互式使用时会请求权限确认，可用 --dangerously-skip-permissions 跳过
  • MCP 服务器显示 "Disconnected" 是正常的，按需连接
  • 插件目录在 ~/.qoderwork/plugins/
