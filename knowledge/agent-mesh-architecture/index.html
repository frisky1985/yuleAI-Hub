  • з»ҹдёҖдёҠдёӢж–Ү вҖ” е…ұдә«дёҠдёӢж–Үз©әй—ҙпјҢйҒҝе…ҚдҝЎжҒҜеӯӨеІӣ
  • зғӯжҸ’жӢ” вҖ” Agent еҸҜд»ҘйҡҸж—¶еҠ е…ҘжҲ–зҰ»ејҖзҪ‘з»ң

ж ёеҝғз»„д»¶

1. MCP Bridge v2пјҲж¶ҲжҒҜи·Ҝз”ұеұӮпјү

дҪңдёәж•ҙдёӘ Mesh зҪ‘з»ңзҡ„дёӯеҝғжһўзәҪпјҢиҙҹиҙЈпјҡ

  • е·Ҙе…·жіЁеҶҢ вҖ” жҜҸдёӘ Agent еҗ‘ Bridge жіЁеҶҢиҮӘе·ұзҡ„иғҪеҠӣпјҲе·Ҙе…·еҲ—иЎЁпјү
  • ж¶ҲжҒҜи·Ҝз”ұ вҖ” ж №жҚ®зӣ®ж Ү Agent еҗҚз§°пјҢе°ҶиҜ·жұӮиҪ¬еҸ‘еҲ°жӯЈзЎ®зҡ„зӣ®зҡ„ең°
  • д»»еҠЎйҳҹеҲ— вҖ” ејӮжӯҘд»»еҠЎзҡ„з®ЎзҗҶе’ҢзҠ¶жҖҒиҝҪиёӘ
  • дёҠдёӢж–ҮеӯҳеӮЁ вҖ” е…ұдә«дёҠдёӢж–Үзҡ„иҜ»еҶҷжҺҘеҸЈ

2. Agent иҠӮзӮ№

  • OpenClaw вҖ” еүҚз«Ҝе…ҘеҸЈ AgentпјҢеӨ„зҗҶз”ЁжҲ·дәӨдә’е’Ңд»»еҠЎеҲҶеҸ‘
  • Hermes вҖ” дё»еҠӣе·ҘдҪң AgentпјҢжү§иЎҢжҗңзҙўгҖҒд»Јз ҒгҖҒж–Үд»¶ж“ҚдҪң
  • QoderCLI вҖ” е·Ҙе…·еһӢ AgentпјҢжҸҗдҫӣзі»з»ҹзә§ж“ҚдҪңе’Ңи„ҡжң¬жү§иЎҢ

йҖҡдҝЎеҚҸи®®

жүҖжңүйҖҡдҝЎйҖҡиҝҮ MCP (Model Context Protocol) иҝӣиЎҢпјҢйҮҮз”Ё JSON-RPC ж јејҸпјҡ

// иҜ·жұӮ
{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "agent-chat",
    "arguments": {
      "agent": "hermes",
      "message": "еҲҶжһҗиҝҷд»Ҫд»Јз Ғ"
    }
  },
  "id": 1
}

// е“Қеә”
{
  "jsonrpc": "2.0",
  "result": {
    "content": [{ "type": "text", "text": "еҲҶжһҗз»“жһң..." }]
  },
  "id": 1
}

ж•°жҚ®жөҒи®ҫи®Ў

дёүз§Қдё»иҰҒзҡ„йҖҡдҝЎжЁЎејҸпјҡ

еҗҢжӯҘи°ғз”Ё

йҖӮз”ЁдәҺеҝ«й—®еҝ«зӯ”пјҲвүӨ8 з§’пјүгҖӮи°ғз”Ёж–№ Agent еҸ‘йҖҒиҜ·жұӮеҗҺйҳ»еЎһзӯүеҫ…пјҢBridge иҪ¬еҸ‘е№¶иҒҡеҗҲе“Қеә”еҗҺиҝ”еӣһгҖӮ

ејӮжӯҘе§”жҙҫ

йҖӮз”ЁдәҺиҖ—ж—¶д»»еҠЎгҖӮи°ғз”Ёж–№з«ӢеҚіж”¶еҲ° task_idпјҢеҗҺз»ӯйҖҡиҝҮ task-result иҪ®иҜўз»“жһңгҖӮ

Agent A вҶ’ Bridge: agent-delegate(hermes, "еҲҶжһҗж—Ҙеҝ—")
Bridge вҶ’ Agent A: { task_id: "abc123" }

Agent A вҶ’ Bridge: task-result("abc123")
Bridge вҶ’ Agent A: { status: "completed", result: "..." }

дёҠдёӢж–Үе…ұдә«

йҖҡиҝҮ context-write/context-read е®һзҺ°и·Ё Agent зҡ„дҝЎжҒҜдј йҖ’гҖӮж•°жҚ®жҢҒд№…еҢ–еңЁ context.json дёӯгҖӮ

и®ҫи®ЎеҶізӯ–и®°еҪ•

еҶізӯ–ж–№жЎҲзҗҶз”ұ
йҖҡдҝЎеҚҸи®®MCP + JSON-RPCж ҮеҮҶеҢ–гҖҒдёҺзҺ°жңүе·Ҙе…·е…је®№гҖҒжҳ“жү©еұ•
зҠ¶жҖҒеӯҳеӮЁжң¬ең°ж–Үд»¶ JSONйӣ¶еӨ–йғЁдҫқиө–гҖҒжҳ“и°ғиҜ•гҖҒжҖ§иғҪи¶іеӨҹ
д»»еҠЎжЁЎеһӢеҗҢжӯҘ+ејӮжӯҘеҸҢжЁЎзҒөжҙ»жҖ§пјҢеҝ«д»»еҠЎдёҚеј•е…ҘејӮжӯҘејҖй”Җ
еҸ‘зҺ°жңәеҲ¶Bridge дёӯеҝғеҢ–жіЁеҶҢз®ҖеҚ•еҸҜйқ пјҢйҖӮеҗҲ Agent ж•°йҮҸдёҚеӨҡзҡ„еңәжҷҜ
