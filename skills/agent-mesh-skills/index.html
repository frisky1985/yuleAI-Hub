Agent Mesh 技能清单

所有 Agent 的角色定义和能力清单

1. Orchestrator

运行于: OpenClaw

通信: agent-chat / agent-delegate

能力:

  • 管道编排：按阶段推进 Harness Engineering 管道
  • 状态管理：维护 orchestrator-state.json
  • 失败接管：3 次重试后自动介入修复
  • 报告输出：生成最终总结报告

入口:

python3 harness-orchestrator.py run "<goal>"
python3 harness-orchestrator.py resume   # 续传
python3 harness-orchestrator.py status   # 状态

2. Hermes

运行于: DeepSeek API（Hermes CLI）

通信: agent-chat / agent-delegate

能力:

  • 需求分析：解析用户需求，输出结构化需求文档
  • 排期：基于需求制定时间线和里程碑
  • 测试用例：为代码创建测试用例
  • 测试执行：运行测试并输出测试报告
  • 快速代码审查

调用:

hermes -z "prompt" --yolo              # 同步
hermes -z "long task" --yolo           # 异步

3. QoderCLI

运行于: DeepSeek API（QoderCLI）

通信: agent-delegate

能力: 架构设计、编码实现、代码审查、代码探索

子角色

角色Qoder Agent职责
Architectingeek-sw-arch系统架构、模块边界、接口定义
Researcheringeek-sw-reviewer代码探索、差距分析、技术债务识别
Planneringeek-sw-arch任务分解、执行顺序、合同定义
Implementeringeek-sw-devTDD 编码实现
Testeringeek-sw-test单元测试、覆盖率、静态分析
Revieweringeek-sw-reviewer代码审查、质量门禁、架构合规检查

调用:

qodercli -p --dangerously-skip-permissions "prompt"          # 默认
qodercli -a ingeek-sw-dev -p "implement this"               # 指定子角色
