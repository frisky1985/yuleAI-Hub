⚡ Hermes 快速操作技巧

📅 2026-05-25 ⏱️ 即用型技巧
Hermes技巧CLI

Hermes Agent 日常使用中最实用的快速操作，一张速查表解决 80% 的场景。

快速启动

命令说明
hermes对话模式启动
hermes -z "问题"单次查询（one-shot），不启动交互
hermes --model openai/gpt-4指定模型启动

常用操作

Web 搜索

直接问，Hermes 自动调用内置搜索工具：

"搜索一下最近的 Python 3.13 新特性"

文件读写

"读取 ~/project/config.json 的内容"
"把这份代码保存到 ~/output/result.py"

代码执行

"运行 python3 -c 'print(\"hello\")'"
"帮我写一个快速排序算法并执行测试"

跨 Agent 调用

通过 MCP Bridge 调用其他 Agent：

"帮我用 agent-chat 问一下 qodercli 现在几点了"

踩坑备忘

  • One-shot 用 -z — Hermes 不支持 -p 参数，使用 -z 进行单次查询
  • 超时增大 timeout — DeepSeek API 高峰时响应慢，增大 gateway_timeout
  • MCP 连接失败 — 确保 Bridge 的 Python 版本与 Hermes 兼容（Hermes 使用 venv Python 3.11）
