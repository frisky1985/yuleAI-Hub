Using Superpowers

Superpowers 技能的使用指引和最佳实践。

Superpowers 是一个 spec-first、TDD、subagent-driven 的开发工作流。

适用于：

  • 构建新功能或应用 → 触发调研→计划→子代理执行循环
  • 调试 bug 或测试失败 → 触发系统化根因分析流程
  • 用户说"让我们构建"、"帮我计划"、"我要添加 X"、"这个坏了"
  • 完成功能分支 → 测试验证 + 合并/PR 选项
