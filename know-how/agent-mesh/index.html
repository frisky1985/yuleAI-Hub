Agent Mesh — 跨 Agent 协同网络搭建

2026-05-26 · #MCP #Agent协同 #OpenClaw #Hermes #QoderCLI

让 OpenClaw、Hermes、QoderCLI 三个 Agent 通过 MCP Bridge 互相协作。

架构

                ┌──────────────────────┐
                │    MCP Bridge v2     │
                │  (mcp_bridge_server) │
                └────┬──────┬──────┬───┘
                     │      │      │
               ┌─────┘  ┌───┴───┐  └──────┐
               ▼        ▼       ▼         ▼
         ┌─────────┐ ┌──────┐ ┌────────┐ ┌─────────┐
         │OpenClaw │ │Hermes│ │QoderCLI│ │  共享    │
         │         │ │      │ │        │ │ 工作区   │
         └─────────┘ └──────┘ └────────┘ └─────────┘

MCP 工具一览

工具说明响应
agent-status检查所有 agent 状态<1s
agent-chat快速对话≤8s
agent-delegate委派异步任务即时返回 task_id
task-result查询任务结果即时
task-list查看最近任务即时
context-read读取共享上下文即时
context-write写入共享上下文即时

安装

# 配置 OpenClaw
openclaw mcp set mcp-bridge '{
  "command": "python3",
  "args": ["/path/to/mcp_bridge_server.py"]
}'

# 验证
openclaw mcp list
