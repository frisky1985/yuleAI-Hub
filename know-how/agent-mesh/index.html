е·Ҙе…·иҜҙжҳҺе“Қеә”
agent-statusжЈҖжҹҘжүҖжңү agent зҠ¶жҖҒ<1s
agent-chatеҝ«йҖҹеҜ№иҜқпјҲеҗҢжӯҘпјҢйҖӮеҗҲеҝ«й—®еҝ«зӯ”пјүвүӨ8s
agent-delegateе§”жҙҫејӮжӯҘд»»еҠЎпјҲз«ӢеҚіиҝ”еӣһ task_idпјүеҚіж—¶
task-resultжҹҘиҜўејӮжӯҘд»»еҠЎз»“жһңеҚіж—¶
task-listжҹҘзңӢжңҖиҝ‘е§”жҙҫзҡ„д»»еҠЎеҚіж—¶
context-readиҜ»еҸ–е…ұдә«дёҠдёӢж–ҮеҚіж—¶
context-writeеҶҷе…Ҙе…ұдә«дёҠдёӢж–ҮеҚіж—¶

дҪҝз”ЁжөҒзЁӢ

ж–№ејҸдёҖпјҡеҝ«йҖҹй—®йўҳ

"её®жҲ‘з”Ё agent-chat й—® hermesпјҡзҺ°еңЁжҳҜеҮ зӮ№пјҹ"
вҶ’ 8 з§’еҶ…еҫ—еҲ°еӣһеӨҚ

ж–№ејҸдәҢпјҡиҖ—ж—¶д»»еҠЎ

жӯҘйӘӨ1 - "з”Ё agent-delegate жҠҠд»»еҠЎдәӨз»ҷ hermesпјҡжҹҘжҳҺеӨ©иҲӘзҸӯ"
жӯҘйӘӨ2 - еҮ з§’еҗҺ "з”Ё task-result жҹҘдёҖдёӢз»“жһң"

ж–№ејҸдёүпјҡе…ұдә«дёҠдёӢж–Ү

жӯҘйӘӨ1 - "з”Ё context-writeпјҢkey=project_root, value=/myapp"
жӯҘйӘӨ2 - е…¶д»– agent "з”Ё context-read жҹҘ project_root"

еҚҸеҗҢи§„еҲҷ

  1. и°Ғз”ЁйғҪиЎҢ вҖ” дёүдёӘ Agent д»»ж„ҸдёҖдёӘйғҪиғҪйҖҡиҝҮ Bridge и°ғз”Ёе…¶д»– Agent
  2. ејӮжӯҘдјҳе…Ҳ вҖ” иҖ—ж—¶еӨ§дәҺ 8 з§’зҡ„д»»еҠЎз”Ё agent-delegateпјҢдёҚзЎ¬зӯү
  3. дёҠдёӢж–Үе…ұдә« вҖ” йҮҚиҰҒдҝЎжҒҜз”Ё context-write еӯҳпјҢе…¶д»– Agent з”Ё context-read жӢҝ
  4. зҠ¶жҖҒйҖҸжҳҺ вҖ” task-list йҡҸж—¶жҹҘзңӢжүҖжңүиҝӣиЎҢдёӯзҡ„д»»еҠЎ
  5. еӨұиҙҘйҮҚиҜ• вҖ” еҰӮжһң Agent и¶…ж—¶жҲ–жҠҘй”ҷпјҢжҚўдёҖдёӘ Agent еҶҚиҜ•
