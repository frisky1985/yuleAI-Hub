Agent Mesh — 跨 Agent 协同网络搭建

2026-05-26

MCPAgent协同OpenClawHermesQoderCLI

让 OpenClaw、Hermes、QoderCLI 三个 Agent 通过 MCP Bridge 互相协作。任意一个 Agent 都可以把任务委派给其他 Agent，共享上下文。

📚 相关参考：架构设计 · 协同规则

架构概览

                ┌──────────────────────┐
                │    MCP Bridge v2     │
                │  (mcp_bridge_server) │
                └────┬──────┬──────┬───┘
                     │      │      │
               ┌─────┘  ┌───┴───┐  └──────┐
               ▼        ▼       ▼         ▼
         ┌─────────┐ ┌──────┐ ┌────────┐ ┌─────────┐
         │OpenClaw │ │Hermes│ │QoderCLI│ │  共享    │
         │         │ │      │ │        │ │ 工作区   │
         └─────────┘ └──────┘ └────────┘ └─────────┘

安装

# 1. 确认三个 agent 都已安装
openclaw --version
hermes --version
qodercli --version

# 2. 下载 MCP Bridge
mkdir -p ~/.openclaw/mcp-bridge
curl -sL https://raw.githubusercontent.com/frisky1985/yuleAI-Hub/main/tools/mcp-bridge/mcp_bridge_server.py \
  -o ~/.openclaw/mcp-bridge/mcp_bridge_server.py

# 3. 注册到 OpenClaw
openclaw mcp set mcp-bridge '{
  "command": "python3",
  "args": ["~/.openclaw/mcp-bridge/mcp_bridge_server.py"]
}'

# 4. 验证
openclaw mcp list

MCP 工具一览

工具说明响应
agent-status检查所有 agent 状态<1s
agent-chat快速对话（同步，适合快问快答）≤8s
agent-delegate委派异步任务（立即返回 task_id）即时
task-result查询异步任务结果即时
task-list查看最近委派的任务即时
context-read读取共享上下文即时
context-write写入共享上下文即时

使用流程

方式一：快速问题

"帮我用 agent-chat 问 hermes：现在是几点？"
→ 8 秒内得到回复

方式二：耗时任务

步骤1 - "用 agent-delegate 把任务交给 hermes：查明天航班"
步骤2 - 几秒后 "用 task-result 查一下结果"

方式三：共享上下文

步骤1 - "用 context-write，key=project_root, value=/myapp"
步骤2 - 其他 agent "用 context-read 查 project_root"

协同规则

  1. 谁用都行 — 三个 Agent 任意一个都能通过 Bridge 调用其他 Agent
  2. 异步优先 — 耗时大于 8 秒的任务用 agent-delegate，不硬等
  3. 上下文共享 — 重要信息用 context-write 存，其他 Agent 用 context-read 拿
  4. 状态透明 — task-list 随时查看所有进行中的任务
  5. 失败重试 — 如果 Agent 超时或报错，换一个 Agent 再试

文件结构

~/.openclaw/mcp-bridge/
├── mcp_bridge_server.py    # MCP Bridge v2 服务端
├── workspace/               # 共享工作区
│   ├── context.json         # 共享上下文
│   └── tasks/               # 异步任务结果
└── SKILL.md                 # Skill 文档
