Hermes Agent 使用技巧

2026-05-25

Hermes技巧

Hermes Agent 的日常使用技巧和踩坑记录。

💡 想要快速速查表？看看 ⚡ Hermes 快速操作技巧 的浓缩版。

快速启动

# 对话模式
hermes

# 单次查询（one-shot）
hermes -z "你的问题"

关键技巧

1. Web 搜索

Hermes 内置 web 搜索能力，直接问即可：

"搜索一下最近的 Python 3.13 新特性"

2. 文件读写

Hermes 可以直接读写文件系统中的文件：

"读取 ~/project/config.json 的内容"
"把这份代码保存到 ~/output/result.py"

3. 代码执行

"运行 python3 -c 'print(\"hello\")'"
"帮我写一个快速排序算法并执行测试"

4. MCP 工具集成

通过 MCP Bridge 可以调用 OpenClaw 和 QoderCLI：

"帮我用 agent-chat 问一下 qodercli 现在几点了"

踩坑记录

  • One-shot 模式不输出 — 使用 -z 而不是 chat -p，Hermes 不支持 -p 参数
  • API 超时 — DeepSeek API 有时响应慢，增大 timeout 设置，或在 Bridge 中使用异步模式
  • MCP 连接失败 — 确保 Bridge 的 Python 版本与 Hermes 兼容（Hermes 使用 venv 中的 Python 3.11）
