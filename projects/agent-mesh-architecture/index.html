设计原则

  1. MCP 优先 — 所有跨 Agent 通信走 MCP 协议，不引入额外依赖
  2. 异步委派 — 耗时任务异步执行，立即返回 task_id
  3. 共享上下文 — 通过 context-read/write 实现状态共享
  4. 零外部依赖 — 纯 Python 标准库，无需 pip install
  5. 即插即用 — 配置一个 MCP server 即可加入网络

通信流程

  Agent A                    MCP Bridge                  Agent B

    │◄── {status:"completed"} ──│                          │

技术栈

  • Python 3.9+ 标准库
  • MCP stdio 传输协议（JSON-RPC 2.0 + Content-Length 帧）
  • 多线程后台任务执行
  • JSON 文件持久化任务状态和共享上下文
