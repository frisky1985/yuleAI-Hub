📁 yuleASR-Configurator 项目概览

📅 2026-05-27 📁 项目 · AUTOSAR 配置工具
yuleASRReactAUTOSAR配置工具

yuleASR-Configurator 是一个面向 AUTOSAR 嵌入式软件开发的 BSW 模块图形化配置工具。本文档从全局视角描述项目的技术栈、架构和关键信息。

技术栈

层技术版本
前端框架React19
构建工具Vite7
CSS 框架Tailwind CSS3
状态管理zustand—
路由React Router—
后端Fastify—
ORMDrizzle—
数据库PostgreSQL—

BSW 模块

项目包含 37 个 BSW 模块，覆盖 AUTOSAR 基础软件的核心类别：

  • 系统服务层：OS、EcuM、BswM、SchM
  • 存储服务层：NvM、Fee、Ea
  • 通信服务层：Can、Lin、Ethernet 协议栈
  • I/O 服务层：Dio、Port、Adc、Pwm、ICU、Gpt
  • 诊断和安全层：Dcm、Dem、Fim、WdgM
  • 其他基础模块

路由结构

/                       # 首页重定向
├── /configurator/      # 配置器主应用
│   ├── /               # 仪表盘 / 模块概览
│   ├── /modules        # 模块列表
│   ├── /module/:id     # 单个模块配置（Editor）
│   └── /preview        # 配置预览
├── /community/         # 社区模块
│   ├── /               # 社区首页
│   ├── /forum          # 论坛
│   └── /docs           # 文档
└── /api/               # 后端 API 路由

架构说明

项目采用 monorepo 多包结构：

yuleASR-Configurator/
├── packages/
│   ├── frontend/       # React 前端应用
│   ├── backend/        # Fastify 后端服务
│   ├── shared/         # 共享类型定义和工具
│   └── database/       # Drizzle schema 和迁移
├── package.json        # monorepo 根配置
└── pnpm-workspace.yaml
