# yuleAI-Hub — AI Agent 知识沉淀与分享

> 记录和分享 AI Agent 协同的 Know-How。

## 结构

```
/
├── index.html              # 首页
├── search.json             # 搜索索引
├── know-how/               # 知识文章
│   ├── agent-mesh/         # 跨 Agent 协同
│   ├── hermes-tips/        # Hermes 技巧
│   └── ...
├── tools/                  # 工具配置指南
├── projects/               # 项目文档
├── about/                  # 关于
├── assets/css/style.css    # 样式
└── _layouts/default.html   # 布局模板
```

## 使用

1. Fork 本仓库
2. 开启 GitHub Pages（Settings → Pages → main branch / root）
3. 在对应目录下添加 .md 或 .html 文件
4. 更新 `search.json` 添加搜索索引

## 特性

- ✨ 纯静态，GitHub Pages 免费托管
- 🌗 支持深色模式
- 🔍 客户端全文搜索
- 📱 响应式设计
- 🚀 零依赖，即写即用
