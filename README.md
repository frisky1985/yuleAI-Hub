# 🧠 yuleAI-Hub — AI 知识家园

> 个人 AI 知识家园 · 学习 · 记录 · 分享

记录和分享 AI Agent 学习与实践中的思考、经验和 Know-How。

## 站点结构

```
/
├── index.html              # 首页 — 知识仪表盘
├── search.json             # 搜索索引
├── knowledge/              # 📚 知识库 — 系统性理论知识
├── know-how/               # 📖 Know-How — 实践指南
├── skills/                 # ⚡ Skills — 原子化技巧
├── rules/                  # 📋 规则集 — 公约规范
├── tools/                  # 🔧 工具 — 配置指南
├── projects/               # 📁 项目 — 项目文档
├── about/                  # 💬 关于 · 社区
├── _layouts/default.html   # 布局模板
└── assets/css/style.css    # 样式
```

## 内容栏目

| 栏目 | 定位 | 内容特征 |
|------|------|----------|
| 📚 **知识库** | 系统性、理论性 | 长文、结构完整 |
| 📖 **Know-How** | 实践性、经验性 | 中篇、步骤清晰 |
| ⚡ **Skills** | 原子化、可复用 | 短篇、即插即用 |
| 📋 **规则集** | 约束性、规范性 | 清单、规范文档 |
| 🔧 **工具** | 配置指南 | 工具安装配置 |
| 📁 **项目** | 项目文档 | 架构部署笔记 |

## 使用

1. Fork 本仓库
2. 开启 GitHub Pages（Settings → Pages → main branch / root）
3. 在对应目录下添加 .html 或 .md 文件
4. 更新 `search.json` 添加搜索索引

## 特性

- ✨ 纯静态，GitHub Pages 免费托管
- 🌗 支持深色模式
- 🔍 客户端全文搜索
- 📱 响应式设计
- 🚀 零依赖，即写即用
